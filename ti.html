The Loot Box: Software de Gestão

Formação Técnica Profissional (T.I.) | 4º Bimestre

Desenvolvimento de uma aplicação desktop (CRUD) para gerenciar o estoque e as vendas de uma pequena loja de jogos e colecionáveis fictícia, "The Loot Box".

Um sistema robusto para cadastro, leitura, atualização e exclusão de produtos, clientes e vendas.

Visão Geral do Software

O projeto foi o trabalho de conclusão da Formação Técnica, com o objetivo de consolidar os conceitos de programação orientada a objetos, banco de dados e desenvolvimento de interface gráfica. A aplicação foi projetada para ser intuitiva e funcional, resolvendo um problema de negócio real.

Tecnologias Utilizadas

  • Linguagem: Python (usando a biblioteca Tkinter para a GUI).
  • Banco de Dados: SQLite para armazenamento local dos dados.
  • Controle de Versão: Git e GitHub para gerenciamento do código-fonte.

Funcionalidades Principais

O software foi entregue com os seguintes módulos operacionais:

  • Gestão de Produtos: Adicionar, editar, buscar e remover jogos do estoque.
  • Cadastro de Clientes: Gerenciamento de clientes e histórico de compras.
  • Módulo de Vendas: Um Ponto de Venda (PDV) simples para registrar novas transações e atualizar o estoque automaticamente.
  • Relatórios: Geração de relatórios básicos (ex: produtos mais vendidos, estoque baixo).

Este projeto foi fundamental para entender o ciclo de vida completo do desenvolvimento de software, desde a concepção e modelagem do banco de dados até a implementação da interface do usuário.
